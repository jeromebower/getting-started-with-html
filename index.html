HTML Markup
Tags - contained in angled brackets. Most contain an opening tag and a closing tag, some do not require a closing tags called void tags.
Element - refers to everything in a set of opening tags and closing tags
Attribute - property of an HTML element

Tag
<name>contents</name> = element
<name> = Opening tag
</name> = Closing tag

Examples
<b></b> = bold
<em></em> = emphasis(italics)

Attributes
<tag attr="value">contents</tag>

Example
<a href="www.redit.com">derp</a>
href = attribute name
"www.reddit.com" = value
